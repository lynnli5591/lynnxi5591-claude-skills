# lynnxi5591-claude-skills
从业务需求到可交互HTML原型的完整链路的产品设计技能。
版本不断迭代，使用最新版！
V1.0存在的问题：
1、原型不能在沙箱中渲染的问题（经过 5 次迭代，发现在artifacts沙箱环境中受限，生成和渲染原型图的效果没有在对话框中渲染的效果好）
2、不能生成移动端原型的问题（因为它通过我的实战案例默认生成 Web 端了，没有注意用户的需求优先级，用户通常不会注意要描述这个需求）
3、不能下载 html 原型的问题

V2.0  更新：
1、完善：产品终端类型（4 种）必选，可选【需求业务领域】
2、增加：支持多次修改 产品架构方案
3、去掉：在artifacts 应用环境中渲染原型图的功能，Artifact 负责收集需求和规划架构，生成原型这一步改回触发对话框生成，让 Claude 在对话里直接输出高质量的可交互原型
